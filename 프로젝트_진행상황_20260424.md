#### 시도 4 — HTML `greenFeeWD` 직접 파싱 (커밋: 431c3da)
- 결과: 51개 전부 "가격 미등록" → 실패

#### 진단 (디버그 로그 추가해서 확인)
- Status 200 ✅, HTML 170K~212K 크기 ✅, Cloudflare 차단 아님 ✅
- **`greenFee` 문자열은 있지만 패턴이 안 맞음** → 패턴 확인 필요

#### 2차 진단 — 주변 문자열 출력
**HTML 실제 형태 (이스케이프된 JSON):**
```
\"greenFeeWD\":\"5000\",\"greenFeeWE\":\"6500\"
\"course\":{\"id\":\"55ae054da5fc5513e1208eb2\",\"name\":\"ALPINE GOLF CLUB\"
```

**원인:** Next.js RSC 스트리밍에서 JSON이 이스케이프된 형태(`\"`)로 들어있음

**해결책 (작업 지시서 작성 완료, 아직 적용 전):**
정규식에 `\\?`를 추가해서 이스케이프된 따옴표와 일반 따옴표 둘 다 매칭하도록 수정

```js
// 변경 전:
const wdMatch = html.match(/"greenFeeWD"\s*:\s*"([\d,]+)"/);

// 변경 후:
const wdMatch = html.match(/\\?"greenFeeWD\\?"\s*:\s*\\?"([\d,]+)\\?"/);
```

**다음 단계:** 마틴이 이 수정을 Claude Code로 적용하고 테스트 후 결과 확인 필요

## 8. Golfdigg 페이지 HTML 구조 (중요 참고)

### 실제 HTML에 박혀있는 형태 (Next.js RSC 스트리밍)
```
\"course\":{\"id\":\"55ae054da5fc5513e1208eb2\",\"name\":\"ALPINE GOLF CLUB\",...
\"greenFeeWD\":\"5000\",\"greenFeeWE\":\"6500\"
\"seoTitle\":\"ALPINE GOLF CLUB | 알파인 골프 클럽\"
```

### 올바른 정규식
```js
/\\?"greenFeeWD\\?"\s*:\s*\\?"([\d,]+)\\?"/
/\\?"course\\?"\s*:\s*\{[^}]*?\\?"name\\?"\s*:\s*\\?"([^"\\]+)\\?"/
```

`\\?` = "백슬래시가 있든 없든 매칭" (이스케이프된 `\"` 와 일반 `"` 둘 다 OK)
