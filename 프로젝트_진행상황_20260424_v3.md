## 3. 이번 세션에서 학습한 Golfdigg API 명세

### A. 코스 리스트 API
```
GET https://api3.golfdigg.com/api/course/all/pageable?page=0&size=50
```
- 응답: `{ content: [...], totalElements: 141, totalPages: 8 }`
- 태국 전체 141개 (2026-04-24 기준)

### B. 슬롯 API (실제 가격)
```
GET https://api.golfdigg.com/golfdigg/slot/v4/list/?courseId=<ID>&date=<YYYY-MM-DD>
```
- 응답: `{ success: true, status: 0, model: { slots: [...], notes, bookingPeriod } }`
- `slots` 배열 내 객체 주요 필드:
  - `startDateString`: "2026-04-27T07:38:00+0700" (시간대 파싱용)
  - `priceCategory`: "NORMAL" | "SPECIAL"
  - `price.price`: 그린피 (freeCart/freeCaddy에 따라 의미 다름)
  - `freeCart`, `freeCaddy`: true면 price에 이미 포함
  - `cartSetting`: `{ require, priceOriginal, priceSale }`
  - `caddySetting`: `{ priceOriginal, priceSale, priceBase }`
  - `status`: "OPEN" 등 (부킹 가능 여부 — 가격과 무관)

### C. 올인가격 계산 공식
```
total = price.price
      + (!freeCart  ? cart.priceSale ?? cart.priceOriginal ?? 0 : 0)
      + (!freeCaddy ? caddy.priceSale ?? caddy.priceOriginal ?? caddy.priceBase ?? 0 : 0)
```

### D. 시간대 판정
- 오전: `startDateString`의 hour가 06~10
- 오후: hour가 11~15
- 태국 평일은 오전/오후 구분 거의 없음 → 평일은 오전값만 취해서 am=pm 동일 저장
- 주말은 오전/오후 각각 수집
